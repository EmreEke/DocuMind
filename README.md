# DocuMind - AI-Powered Document Search & RAG System

Bu proje, BİL440 YZ Destekli Yazılım Geliştirme dersi final ödevi kapsamında geliştirilmiştir.

## Proje Hakkında
DocuMind, kullanıcıların PDF dokümanları yükleyebildiği, dokümanlar üzerinde anlamsal (vektörel) arama yapabildiği ve yapay zeka destekli sorular sorabildiği bir RAG (Retrieval-Augmented Generation) sistemidir.

## Teknoloji Yığını
- **Frontend:** React + Vite
- **Backend:** Python FastAPI
- **AI Engine:** LangChain + Google Gemini 2.5 Flash
- **Veritabanı:** PostgreSQL + pgvector (Dockerized)
